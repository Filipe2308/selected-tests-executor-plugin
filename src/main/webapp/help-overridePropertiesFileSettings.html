In this section you can set or override your properties.
If you click on the "Load Settings From Properties File" button it will fill the properties bellow from the properties file.
If you click on the "Show/Hide Available Fields" button it will show/hide all the fields used in the tests you defined in the properties file. Next to each Available field, for "Show Fields", appears (howManyTestsHasThisField/amountOfTests).
